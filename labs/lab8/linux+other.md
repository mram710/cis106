# What is Linux?

Linux is a Unix-like Operating System popular in academic and business enivornments.

# 3 Reasons to use Linux:

Reason 1: Linux is free compared to other OS's such as Windows.

Reason 2: Linux is open source which allows you to customize the source code if you choose to do so.

Reason 3: Linux is more stable compared to Windows OS so it is less prone to crashing.

# Other operating systems
* **Microsoft Windows** is a group of several proprietary graphical operating system families, all of which are developed and marketed by Microsoft. 
* **macOS** is a series of proprietary graphical operating systems developed and marketed by Apple Inc. 
* **FreeBSD** is a free and open-source Unix-like operating system descended from the Berkeley Software Distribution (BSD), which was based on Research Unix.
* **RISC OS** is a computer operating system originally designed by Acorn Computers Ltd in Cambridge, England.
