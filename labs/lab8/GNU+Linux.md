# What is GNU+Linux?

GNU+Linux is a Unix-like Operating System popular in academic and business enivornments.
![LinuxLogo](https://rb.gy/zvbetm)


# 3 Reasons to use GNU+Linux:

Reason 1: GNU+Linux is free compared to other OS's such as Windows.

Reason 2: GNU+Linux is open source which allows you to customize the source code if you choose to do so.

Reason 3: GNU+Linux is more stable compared to Windows OS so it is less prone to crashing.

# Other operating systems
