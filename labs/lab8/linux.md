# What is Linux?

Linux is a Unix-like Operating System popular in academic and business enivornments.

# 3 Reasons to use Linux:

Reason 1: Linux is free compared to other OS's such as Windows.

Reason 2: Linux is open source which allows you to customize the source code if you choose to do so.

Reason 3: Linux is more stable compared to Windows OS so it is less prone to crashing.
